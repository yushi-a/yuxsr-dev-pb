# yuxsr-dev-pb

## 概要 (Overview)

このリポジトリは `yuxsr-dev` プロジェクトで使用される Protocol Buffers の定義を管理します。
`gencode.sh` スクリプトを通じて、`proto/` ディレクトリ内の定義から Go と TypeScript のクライアント/サーバコードを生成します。

生成されるコードは、マイクロサービス間の gRPC (Connect) 通信での利用を想定しています。

## 依存 (Dependencies)

コード生成スクリプトを実行するには、以下のツールがローカル環境にインストールされている必要があります。

### 1. Node.js & npm
TypeScript のコンパイルや、コード生成に必要な npm パッケージの管理に使用します。
以下のコマンドで、必要なパッケージをインストールしてください。

```bash
npm install
```

### 2. Go
生成された Go コードのモジュールを初期化するために Go のツールチェーンが必要です。
`gencode.sh` スクリプトが内部で `go mod` コマンドを実行します。

お使いの環境にGoをインストールしてください。

## 使用方法 (Usage)

### コード生成 (Code Generation)

リポジトリのルートで以下のコマンドを実行すると、`gencode.sh` スクリプトが実行され、コード生成から各言語のセットアップまでの一連のプロセスが開始されます。

```bash
npm run generate
```

生成されたコードは `gencode/` ディレクトリ以下に出力されます。
